# TOPIK Interactive Test

## GitHub Pages

Use `index.html` as the main file. The images for questions 15–16 are already embedded inside `index.html`, so the page works even without external image links. The `assets/` folder is included as a backup copy of the pictures.

## Important

This is a static HTML test. Results are stored in the browser using `localStorage`. That means the teacher mode can see Vanya/Masha results only on the same browser/device where they answered. If students open the GitHub link on their own devices, their answers will not automatically appear in the teacher's browser. For remote collection you need a backend such as Google Sheets, Firebase, or Supabase.
